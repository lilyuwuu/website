Prerequisites

To install this mod, you must have:
  • A copy of Taiko no Tatsujin: Dokodon! Mystery Adventure for the Nintendo 3DS, whether physical or digital
  • A Nintendo 3DS running Luma3DS 7.0 or greater from a modern CFW environment (at a minimum, arm9loaderhax, but everypony should already be using boot9strap TBH)
It is also highly recommended to back up your save data, and then wipe your save file before applying this mod. Existing songs are replaced via LayeredFS, so scores will no longer properly correlate. DLC is unaffected by this mod.

What you need

  • The latest release of Taiko MLP Mod

Installation instructions

  1. Power off your 3DS
  2. Insert your SD card into your computer
  3. Create a folder named titles in the luma folder if it doesn't already exist
  4. Create a folder named 0004000000190E00 in luma -> titles if it doesn't already exist
  5. Copy the contents of the taiko-mlp folder inside of the taiko-mlp-mod .zip to the luma -> titles -> 0004000000190E00 folder on your SD card
  6. Insert your SD card into your 3DS
    • Enable game patching if you haven't already through Luma's configuration menu (accessible by holding SELECT while turning on the 3DS)
The mod should now be properly applied!
